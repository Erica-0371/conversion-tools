# Chat Viewer

Claude Code 对话查看器。将导出的 JSON/JSONL 对话文件转换为阅读友好的 HTML 格式。

## 使用方式

双击 `index.html` 即可打开，拖拽或点击选择对话文件。

## 支持格式

- `.json` — Claude Code 导出的对话 JSON
- `.jsonl` — 每行一条消息的 JSONL 格式
- `.txt` — 文本格式的对话记录

支持同时加载多个文件。

## 功能

- 对话列表侧边栏，支持搜索、日期筛选、排序
- 消息按角色（用户/助手/工具）分色显示
- 单条对话导出为独立 HTML 文件
- 批量选择下载多条对话
- 随时追加更多对话文件

## 隐私

所有数据仅在浏览器本地处理，不上传任何服务器，断网也能用。
